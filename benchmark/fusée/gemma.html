Voici un simulateur de fusée complet contenu dans un seul fichier HTML. Ce simulateur utilise l'API **Canvas HTML5** pour le rendu et un moteur physique simple pour gérer l'accélération, la gravité et le changement d'arrière-plan (de la Terre vers le vide spatial). Copiez le code suivant et enregistrez-le sous le nom `simulateur_fusee.html`. ```html Simulateur de Vol Spatial
Altitude: 0 km
Vitesse: 0 km/s
SORTIE DE L'ATMOSPHÈRE !
``` ### Comment utiliser ce simulateur : 1. Copiez le code ci-dessus. 2. Créez un nouveau fichier sur votre ordinateur et nommez-le `fusée.html`. 3. Ouvrez ce fichier avec n'importe quel navigateur web (Chrome, Firefox, Edge). 4. Cliquez sur le bouton **"LANCER LA FUSÉE"**. ### Détails techniques du simulateur : * **Physique :** Le programme calcule la vitesse en ajoutant une poussée constante moins la gravité : $$\text{Accélération} = \text{Poussée} - \text{Gravité}$$ * **Effet Visuel :** * Le ciel passe dynamiquement du **bleu clair** au **noir profond** à mesure que l'altitude augmente. * Les étoiles apparaissent et défilent plus rapidement à mesure que la vitesse augmente. * Le sol s'éloigne vers le bas de l'écran pour simuler l'ascension. * **Performance :** Utilisation de `requestAnimationFrame` pour un mouvement fluide à 60 images par seconde.
